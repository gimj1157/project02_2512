# 공군 정신전력교육 AI활용 프로그램

## 프로그램의 목적
부사관후보생과 (준)사관후보생, 현역 및 예비역 장병들을 대상으로 실시중인 정신전력교육의 효과를 증진하고 효율성을 높이기 위함임.

## 사용 기술
- Python
- HTML & CSS
- Java Script

## 구현 내용
- 사전 설문을 통해 교육 대상자의 수준 및 성향별로 분류
- 각 분류별 맞춤 교육 메뉴얼 제공
- 질문이 있을 경우 추천 답변 제공

## 실행 화면
(준비중)

## 느낀 점
- 프로젝트를 진행하며 군사기밀 및 보안과 관련하여 명확한 메뉴얼이 없어 시간이 지체되었으나 방첩사 및 보안담당관 협력으로 해결함.
- 기존 틀을 깨뜨리지 않는 선에서 발전적인 개혁을 이루는 방법과 그 효과를 배움.
-  각종 부서들과 협력하며 커뮤니케이션 및 협동력을 기를 수 있는 좋은 기회가 되었음.
